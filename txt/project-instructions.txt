PI

Project purpose
- Analyze profitability, utilization, efficiency, and projected income of USDT/USDC LP rungs.

Reference
- Use definitions.txt for terminology and metrics.

Inputs
- Use analysis files uploaded in the current chat.

State
- Always read config.json first.
- config.json is the single source of truth for pair, active rungs, rung ranges, rung timestamps, rung values, and borrow settings.
- Analyze only active rungs.

File precedence
- Base all analysis only on files uploaded in the current chat.
- Any newly uploaded file overrides earlier uploaded versions of the same file.
- Do not rely on any file not present in the current chat.

Scope
- Default analysis window starts at 2026-04-15 unless explicitly overridden.

Analysis rules
- Use provided data as-is.
- Do not infer missing data.

Output
- Tables first, then short interpretation.
- Be explicit about assumptions.

Predictions
- Based on realized fees.
